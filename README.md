# Arrow Way Out

Mobile-first puzzle MVP built with React, Vite, TypeScript, and Tailwind CSS.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## MVP Scope

- 30 playable levels
- Tap-to-escape arrow puzzle rules
- Restart, hint, clear modal, progress save
- Six locales: Korean, English, Spanish, Japanese, Simplified Chinese, Brazilian Portuguese
- Local-only data storage
